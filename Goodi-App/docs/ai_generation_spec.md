# 任務範本庫 - AI 生成規格說明

## 給其他 AI 的 Prompt：

```
請為 Goodi 兒童任務管理 App 生成任務範本資料。

## 嚴格規格：

### 年齡分組（必須使用這些範圍）
- [5, 6] = 5-6 歲
- [7, 8] = 7-8 歲
- [9, 10] = 9-10 歲
- [11, 12] = 11-12 歲

### 分類（category 只能是以下三種）
- "生活" = 個人生活習慣（刷牙、穿衣、整理物品等）
- "家務" = 家庭事務（洗碗、擦桌、掃地、晾衣服等）
- "學習" = 學業相關（寫作業、讀書、練習等）

### 圖示（icon 只能使用以下 emoji）
🎒 書包    🌱 植物    🧸 玩具    🍽️ 餐具    👟 鞋子
🧹 掃把    📚 書本    🦷 牙齒    🛏️ 床      🧺 籃子
📖 讀書    ✏️ 鉛筆    ⏰ 時鐘    🏃 跑步    🎨 美術
🎵 音樂    🐕 寵物    🌻 花      🧼 肥皂    👕 衣服
🚿 淋浴    🪥 牙刷    🧽 海綿    🗑️ 垃圾桶  📝 筆記
🎯 目標    🏠 房子    🧥 外套    🎮 遊戲    📱 手機

### 輸出 JSON 格式：
{
  "ageRange": [起始年齡, 結束年齡],
  "keyword": "目標關鍵字",
  "tasks": [
    {
      "text": "任務名稱（簡短，5-10字）",
      "points": 積分（1-5 整數）,
      "category": "生活" | "家務" | "學習",
      "icon": "emoji（從上方列表選擇）",
      "description": "給孩子的鼓勵話（15-30字，正面溫暖）"
    }
  ]
}

### 目標關鍵字清單（每個年齡組都要生成）
1. 責任感
2. 時間管理
3. 整潔習慣
4. 自主性
5. 生活規律
6. 專注力
7. 閱讀習慣
8. 學習動機
9. 創造力
10. 解決問題
11. 同理心
12. 情緒管理
13. 團隊合作
14. 禮貌禮儀
15. 感恩惜福

### 注意事項：
- 每組 5 個任務
- 使用繁體中文
- 任務要適合該年齡段
- 描述要正面、鼓勵、溫暖
- 不要使用注音
```
