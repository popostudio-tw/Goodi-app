# 任務範本庫 - 關鍵字與年齡矩陣

## 年齡分組
| 代碼 | 年齡範圍 | 說明 |
|------|----------|------|
| A | 5-6 歲 | 幼兒園大班～小一 |
| B | 7-8 歲 | 小二～小三 |
| C | 9-10 歲 | 小四～小五 |
| D | 11-12 歲 | 小六～國一 |

---

## 關鍵字清單（共 15 個目標）

### 🏠 生活自理類
1. **責任感** - 對自己的事情負責
2. **時間管理** - 準時完成事情、不拖延
3. **整潔習慣** - 保持環境與個人整潔
4. **自主性** - 不需要提醒就能完成
5. **生活規律** - 作息正常、養成好習慣

### 📚 學習成長類
6. **專注力** - 能專心做一件事
7. **閱讀習慣** - 培養閱讀興趣
8. **學習動機** - 主動學習的意願
9. **創造力** - 動手做、發揮想像力
10. **解決問題** - 遇到困難自己想辦法

### 👨‍👩‍👧‍👦 社交情緒類
11. **同理心** - 理解他人感受
12. **情緒管理** - 控制脾氣、表達情緒
13. **團隊合作** - 跟別人一起完成事情
14. **禮貌禮儀** - 有禮貌、尊重他人
15. **感恩惜福** - 感謝身邊的人事物

---

## 預計生成的範本組合

總共：**4 個年齡段 × 15 個目標 = 60 組範本**

每組範本包含 5 個任務，共 **300 個任務**

---

## AI 生成 Prompt 範本

請使用以下格式請 AI 生成每組任務：

```
請為 [年齡] 歲的孩子，設計 5 個培養「[目標]」的具體任務。

輸出格式請使用 JSON：
{
  "ageRange": [起始年齡, 結束年齡],
  "keyword": "目標關鍵字",
  "tasks": [
    {
      "text": "任務名稱",
      "points": 分數(1-5),
      "category": "生活/家務/學習",
      "icon": "emoji圖示",
      "description": "給孩子的鼓勵描述"
    }
  ]
}
```

### 範例 Prompt：

```
請為 5-6 歲的孩子，設計 5 個培養「責任感」的具體任務。

輸出格式請使用 JSON...
```
