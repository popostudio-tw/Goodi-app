# Q4資料導入指南

## 快速導入步驟

### 步驟1：創建11月JSON檔案
在 `Goodi-app` 目錄下創建 `26Q4-Nov.json`，格式如下：

```json
{
  "2026-11-01": {
    "history": "你提供的11/1歷史內容",
    "animalFact": "你提供的11/1動物冷知識"
  },
  ...繼續到11/30
}
```

### 步驟2：創建12月JSON檔案  
在 `Goodi-app` 目錄下創建 `26Q4-Dec.json`，格式相同。

### 步驟3：執行導入
運行命令：
```bash
node import-q4-remaining.js
```

## 或使用單月導入

修改 `import-q3.js` 第15行：
- 導入11月：改為 `'./26Q4-Nov.json'`
- 導入12月：改為 `'./26Q4-Dec.json'`

然後執行：
```bash
node import-q3.js
```
